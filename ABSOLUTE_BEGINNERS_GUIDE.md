## 🔴 WAY #3: MANUAL SETUP (If wizard doesn't work)

### Step-by-step manual installation

### 3.1 Install Python

1. Open browser: https://www.python.org/downloads/
2. Click: "Download Python 3.x.x"
3. Run the downloaded file
4. **IMPORTANT:** Check ✓ "Add Python to PATH"
5. Click "Install Now"
6. Wait for completion

### 3.2 Install QEMU

1. Open browser: https://qemu.weilnetz.de/w64/
2. Download: `qemu-w64-setup-xxxxx.exe` (the .exe file)
3. Run the downloaded file
4. Click Next > Install > Finish
5. **Restart your computer**

### 3.3 Run AVIOS

1. Go to: `D:\personal_projects\avios-project`
2. Double-click: `run-avios.bat`
3. AVIOS starts in a window!
